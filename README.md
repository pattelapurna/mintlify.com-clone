# mintlify.com-clone

Static clone of the Mintlify.com. It's a UI clone
[![Netlify Status](https://api.netlify.com/api/v1/badges/57d18b6c-8abc-4201-b9ed-7f63fd7c9a52/deploy-status)](https://app.netlify.com/projects/mintlify-ui-ppc/deploys)

## What’s Here
- `index.html` — page markup
- `style.css` — site styles
- `assets/` — images and other static files

## Run Locally
You can open the file directly:
- Double click `index.html`

Or serve it locally (recommended for relative paths and fonts):

```bash
git clone https://github.com/pattelapurna/mintlify.com-clone.git
```

Then visit `/path/to/mintlify.com-clone/index.html`.

## Project Notes
- No build step required.
- No dependencies.
